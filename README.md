# Car2Go Frontend

This is the Angular frontend for my Car2Go college project. It has three sides: User, Owner, and Admin.

## Live Links
- Frontend: https://car2goweb.netlify.app
- Backend API: https://car2go-backend-zbum.onrender.com/api

## Tech Used
- Angular
- TypeScript
- Tailwind CSS

## Run Locally
```bash
npm install
ng serve
```
Then open `http://localhost:4200`.

## Build
```bash
ng build
```
